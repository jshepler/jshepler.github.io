My NGU Idle stuff

  • jshepler.ngu.mods - collection of mods that adds many QoL enhancements, bug fixes, and SOME automation
  • jshepler.ngu.discord - a mod that adds Discord integation, currently just Rich Presence
  • jshepler.ngu.framerate - a mod that adds support to set framerate options
  • jshepler.ngu.hardcore - hardcore mode (you die, game over)
  • jshepler.ngu.permaTC - permanent Troll Challege (trolls spawn every 2 minutes for entire game)
  • nguSteamToKong - a tool to downgrade saves from Steam (1.260) to Kong (1.220)
  • some tools - a few tools, including yet another ratio calculator

Other NGU Idle stuff

  • NGUInjector - despite its name, a tool that automates most the game, including manual combat
  • Sayo's guide - generally considered current meta
  • Gear Optimizer - optimizes gear loadouts and a few additional useful tools
  • Sanc's Giant PP Sheet - popular sheet to project PP/AP/Exp gains
  • Blaze's Junkyard - outdated but many still useful sheets
  • Save Analyzer - useful for things like seeing uncapped P/T in infinity cube
